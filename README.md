# Ahsan — Portfolio

A modern, responsive portfolio landing page built with HTML + Tailwind CSS (CDN).

## Features
- Floating glassmorphism navbar
- Animated hero, about, work (bento grid), skills, and contact sections
- Light / dark mode toggle (persisted via localStorage)
- Scroll reveal animations (respects `prefers-reduced-motion`)
- Fully responsive (320px → 1440px)
- SVG icons only (no emojis)

## Local preview
Open `index.html` directly, or serve it:

```bash
npx serve .
```

## Deploy
Push to GitHub and import the repo at [vercel.com/new](https://vercel.com/new).
